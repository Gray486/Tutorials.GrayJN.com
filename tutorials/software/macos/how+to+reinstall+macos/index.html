Step 1: Get into MacOS recovery mode

Turn off the iMac or MacBook using the power button or clicking the Apple logo in the top right and selecting “Shut Down…”. Now turn back on the computer but while you do continue to hold the Power Button on the computer. A screen will say, “Continue holding for startup options”. Once the screen changes to “Loading Startup Options” you can release the key. Now click the “Options” button as seen in the picture below:

This should bring up something that says “Select a User you know the password for. Click the user and enter the password. Now you will see a screen like this:



Step 2: Installing MacOS

Now click “Reinstall MacOS” and “Continue”. Follow the steps until you get to the part that asks which drive to install macOS on. Click the one called “Macintosh HD” or something similar. Now continue on and install MacOS.

Step 1: Get into MacOS recovery mode

Turn off the iMac or MacBook using the power button or clicking the Apple logo in the top right and selecting “Shut Down…”. Now turn back on the computer but as soon as you do hold the “Option” or “Alt” key. You should now see a screen that looks like the one pictured below:

Now click the “Command” and “R” keys at the same time. This should bring up something that says “Starting Internet Recovery” followed by “This may take a while.” Once it does eventually load you should see a screen that looks like the one below:

(Please note the design will vary based on the computer model)

Step 2: Installing MacOS

Now click “Reinstall Mac OS (X)” and “Continue”. It will show the version of MacOS it will install then click continue again. It will say that your computer's eligibility will be verified and click continue once more. Now we need to agree to the Terms of Service so click agree. Select the drive to install MacOS onto. This will usually be called “Macintosh HD” or similar. Once we do that we can continue through the steps and completely reinstall macos.
